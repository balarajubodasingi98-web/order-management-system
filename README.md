# Order Management System (Full Stack)
A full stack Order Management System built using **Spring Boot, JWT Authentication, MySQL, and Vanilla HTML/CSS/JS**.

## Features
- User & Admin Login (JWT based)
- Role-based Authorization
- Product Management (Admin)
- Order Placement & Tracking (User)
- Admin Order Management
- Secure REST APIs

## Tech Stack
- Backend: Spring Boot, Spring Security, JWT, JPA
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Build Tool: Maven

## How to Run
1. Clone the repository
2. Configure MySQL in `application.properties`
3. Run `mvn spring-boot:run`
4. Open `http://localhost:9090/login.html`
